# 基本概念

<div class="grid grid-cols-2 gap-8">
<div>

**糖尿病（DM）** = 代謝性疾病，血糖長期高於標準值

### 三多一少
- 吃多、喝多、尿多、**體重減輕**

### 數字
- 正常空腹血糖：**60～100 mg/dL**
- 台灣盛行率：18歲以上 **11.1%**
- 為國人第 **5～6 大死因**（約218萬人）

# 胰臟三大激素

<div class="mb-4">
<img src="/images/slide84_5.jpg" class="rounded-lg shadow mx-auto max-h-36" />
</div>

| 激素 | 分泌細胞 | 功能 |
|------|---------|------|
| **胰島素** | β 細胞 | 讓葡萄糖進入細胞利用、儲存 |
| **昇糖素** | α 細胞 | 分解肝醣/脂肪 → 升血糖 |
| **生長抑素** | δ 細胞 | 防止血糖波動過大 |

### 📦 葡萄糖儲存去向

| 位置 | 形式 | 說明 |
|------|------|------|
| **肝臟** | 肝醣 | 約 **60%**，可快速釋回血液 |
| **骨骼肌** | 肌肝醣 | 供肌肉運動，無法釋回血液 |
| **脂肪組織** | 三酸甘油酯 | 多餘轉為脂肪；副產物促粥狀動脈硬化 |
| **大腦** | ❌ 無法儲存 | 只能直接利用血糖 → 低血糖立即影響意識 |

---

# 血糖調節 4 大機制

<div class="grid grid-cols-2 gap-8">
<div>

1. 胰島素 ↔ 昇糖素 **負向回饋**
2. **肝醣的儲存與分解**
3. 低血糖 → 交感 → 腎上腺素 → 肝醣分解
4. 長期低血糖 → 生長激素 + 皮質醇 → 提升產糖

> ⚠️ 大腦只能用葡萄糖，無法直接用肝醣  
> → 低血糖**立即**造成意識改變

</div>

# 糖尿病分類

<div class="grid grid-cols-2 gap-8">
<div>

| 類型 | 特徵 | 好發 |
|------|------|------|
| **第一型（IDDM）** | β細胞被自體免疫破壞 | 兒童/青少年 |
| **第二型（NIDDM）** | 分泌不足 / 接受體功能差 | 40歲以上、肥胖 |
| **妊娠糖尿病** | 懷孕期間 | — |
| **LADA（1.5型）** | 緩慢破壞，易誤診為第二型 | 成人 |

> DM vs DI（尿崩症）：ADH 分泌不足或腎臟無反應  
> 特點：多尿、低比重尿

# 病理生理：核心流程

<div class="grid grid-cols-2 gap-8 items-center">
<div>

<img src="/images/slide90_7.jpg" class="rounded-lg shadow w-full" />

</div>
<div>

```
胰島素不足／抗性
    ↓
葡萄糖無法進細胞 → 高血糖
    ↓
改用脂肪當能量
    ↓
脂肪酸 → 酮體
    ↓
代謝性酸中毒（DKA）
```

**三大連鎖反應：**  
① 高血糖 → 脫水  
② 脂肪代謝 → 酮酸  
③ 蛋白質消耗 → 橫紋肌溶解

</div>
</div>

---

# 連鎖①：高血糖 → 脫水

<div class="text-2xl leading-loose mt-8">

血糖↑ → 葡萄糖溢入尿液（**糖尿**）

↓

滲透壓↑ → **滲透性利尿**（大量排尿）

↓

大量排尿 → **脫水**

↓

嚴重時 → **低血容量性休克**

</div>

---

# 連鎖②：脂肪代謝 → 酮酸中毒

<div class="grid grid-cols-2 gap-8">
<div class="text-lg leading-loose">

胰島素缺乏 → 脂肪分解 → 游離脂肪酸

↓ 肝臟代謝

**酮體**（丙酮、乙酸鹽）→ pH 下降

↓ 身體代償

**Kussmaul 呼吸**（深、快）

↓ 丙酮由肺排出

**水果發酵氣味** 🍎

↓ 酸中毒

K⁺ 外移 → **高血鉀 → 心律不整** ⚡

</div>
<div class="flex items-center justify-center">
<div class="bg-red-50 border-l-4 border-red-500 p-6 rounded-lg text-lg">

🔑 **記住：**
- 呼吸：**又深又快**
- 氣味：**水果發酵**
- 危險：**高血鉀致命**

</div>
</div>
</div>

---

# 連鎖③：蛋白質消耗 → 橫紋肌溶解

<div class="leading-loose">

無胰島素 → 肌肉蛋白分解 → 胺基酸轉葡萄糖（**惡性循環**）

長期後果：免疫力↓ ／ 體重減輕 ／ 器官負擔增加

</div>

<div class="mt-4 bg-red-50 border-l-4 border-red-600 p-4 rounded-lg">

**⬇️ 進一步導致橫紋肌溶解**

</div>

---

# 🔴 橫紋肌溶解（Rhabdomyolysis）

<div class="grid grid-cols-2 gap-6">
<div>

**機制**：骨骼肌破壞 → 肌紅蛋白/K⁺/磷釋入血液

**臨床症狀：**
- 🦵 肌肉：劇烈疼痛、腫脹、無力
- 🟤 **尿液（最具特徵！）**
  - 可樂色 / 茶褐色
  - 大量**泡沫**（蛋白質溢出）
  - 少尿 → 無尿
- 🤒 全身：噁心、嘔吐、發燒、極度虛弱

</div>
<div>

**三大後果：**

| 衝擊 | 結果 |
|------|------|
| 腎小管壞死 | **急性腎衰竭** |
| 高血鉀 | **致命心律不整** |
| 凝血異常 | **DIC** |

<div class="mt-4 bg-amber-50 border-l-4 border-amber-500 p-3 rounded text-sm">

⚠️ 糖尿病誘因：高血糖慢性缺血、蛋白質持續分解、酮酸破壞細胞膜

</div>

</div>
</div>

---

# 長期併發症總覽

<div class="grid grid-cols-2 gap-8 text-lg">
<div>

- ❤️ 心血管疾病、中風
- 🫘 慢性腎臟病
- 🦶 糖尿病足（截肢）
- 👁 視網膜病變（失明）
- 🦵 末梢神經損傷
- 🦠 **容易感染** →

</div>
<div class="flex items-center justify-center">
<div class="bg-gray-50 border rounded-xl p-6 text-center">

**後兩頁詳述：**

🟡 蜂窩性組織炎  
🔴 壞死性筋膜炎

</div>
</div>
</div>

---

# 🟡 感染①：蜂窩性組織炎

<div class="grid grid-cols-2 gap-6">
<div>

**致病機轉**  
皮膚屏障受損  
→ 金黃色葡萄球菌 / 鏈球菌侵入真皮  
→ 高血糖抑制嗜中性球功能

**臨床表現**  
局部 **紅、腫、熱、痛**

</div>
<div>

**嚴重程度**  
- 多數抗生素可治  
- DM患者易**反覆發作**  
- 可進展為**菌血症**

**死亡率**  
- 一般低  
- 延誤或進展為**敗血症**則顯著上升

</div>
</div>

---

# 🔴 感染②：壞死性筋膜炎（NF）

<div class="grid grid-cols-2 gap-6">
<div>

**致病機轉**  
需氧 + 厭氧菌協同感染  
→ 沿筋膜快速蔓延  
→ 血管栓塞 → 組織缺血壞死  
DM神經病變 → 初期**不痛** → 延誤就醫

**特徵**  
皮膚「木頭感」、水泡、壞疽、**捻髮音**

</div>
<div>

<div class="bg-red-100 border-l-4 border-red-600 p-4 rounded-lg mb-4">

**死亡率：20～40%**  
延誤診斷 → **> 70%**

</div>

**⚠️ 院前警覺：**
- 疼痛與外觀**不相符**
- 快速擴大紅斑
- 發燒合併休克

→ 高度懷疑，**立即送院**

</div>
</div>

---

# 🔵 急症①：低血糖

<div class="grid grid-cols-2 gap-8">
<div>

血糖 **< 60 mg/dL**

**症狀進程：**  
顫抖、心悸、冷汗、飢餓  
↓  
頭痛、躁動、抽搐  
↓  
昏迷、**類中風表現**

</div>
<div>

**Whipple's Triad**

<img src="/images/slide96_7.jpg" class="rounded-lg shadow max-h-32 mb-4" />

① 有低血糖症狀  
② 症狀時測到低血糖值  
③ 補糖後症狀緩解

</div>
</div>

<div class="mt-4 bg-blue-50 border-l-4 border-blue-500 p-3 rounded-lg">

⚠️ **任何 DM 患者出現意識改變 → 先懷疑低血糖**

</div>

---

# 🔴 急症②：DKA（糖尿病酮酸血症）

<div class="grid grid-cols-2 gap-8 text-lg">
<div>

胰島素缺乏 → 脂肪代謝 → 酮酸 → 酸中毒

**代償：Kussmaul 呼吸**（深、快）

**特徵：丙酮味**（水果發酵味）

死亡率：**9～14%**

</div>
<div class="flex items-center justify-center">
<div class="bg-red-50 border rounded-xl p-6">

🔑 **DKA 意識改變少見！**

若有意識改變 → 排除：
- 頭部外傷
- 中風
- 藥物中毒

</div>
</div>
</div>

---

# ⚡ 急症③：euDKA — 新型陷阱

<div class="grid grid-cols-2 gap-8">
<div>

血糖**正常**（< 250 mg/dL）  
但有**酮酸中毒**

**常見原因：**  
SGLT-2 抑制劑  
（Jardiance、Forxiga）

→ 葡萄糖從尿排出  
→ 血糖正常但仍燒脂肪產酸

</div>
<div class="flex items-center justify-center">
<div class="bg-yellow-50 border-l-4 border-yellow-500 p-6 rounded-lg">

**症狀：**  
Kussmaul 呼吸  
丙酮味  
噁心、嘔吐

---

⚠️ **血糖正常**  
→ 容易**漏診**  
→ 須高度警覺

</div>
</div>
</div>

---

# 🟠 急症④：HHS（高滲透壓高血糖狀態）

<div class="grid grid-cols-2 gap-8">
<div>

舊稱：NKHS / HHNS  
（現統一用 **HHS**）

胰島素「夠用」避免酮酸  
但不夠讓組織用糖

**臨床三聯：**
- 嚴重**脫水**
- **意識不清**
- 電解質嚴重異常

好發：**老年第二型 DM**

</div>
<div class="flex items-center justify-center">
<div class="bg-orange-50 border-l-4 border-orange-500 p-6 rounded-lg text-2xl text-center">

死亡率

# 10～50%

**所有急症中最高！**

</div>
</div>
</div>

# 📊 四大急性併發症比較（總整理）

<div class="mb-4">
<img src="/images/slide99_9.png" class="rounded-lg shadow mx-auto max-h-32" />
</div>

| | 低血糖 | DKA | **euDKA** | **HHS** |
|--|--------|-----|-----------|---------|
| 形成時間 | 分鐘 | 12～48h | 數小時～數日 | 數日 |
| 血糖 | <60 | >350 | **<250** | >600 |
| 皮膚 | 溼冷 | 乾暖 | 乾暖 | 乾暖 |
| 呼吸 | 正常 | Kussmaul | Kussmaul | 淺快 |
| 氣味 | 正常 | 丙酮味 | 丙酮味 | 正常 |
| 酮體 | 無 | ✅ | ✅ | 無/極少 |
| 意識改變 | 常見 | 少見 | 少見 | 常見 |
| 死亡率 | 低 | 9～14% | 低 | **10～50%** |

# 🚑 院前血糖監測

<div class="grid grid-cols-2 gap-8">
<div>

### 消防署標準
1. 解釋原因
2. 消毒選定部位
3. 採血
4. 加壓止血
5. 棄置針頭

> 適應症：急性意識改變（GCS < 14）

</div>
<div>

### 實務建議
1. 解釋病情
2. 備材（採血針、試紙、紙膠、酒棉）
3. 選定位置消毒
4. 扎針與採血
5. 加壓止血
6. 必要時給糖

> 差異：備物順序、部位未強制、針具回收

</div>
</div>

# 🚑 院前給糖操作

| 情境 | 處置 |
|------|------|
| 意識清楚、無嘔吐 | 口服含糖食物 / 果汁 |
| 意識不清 | 糖粉塗牙齦或舌下 |
| **EMTP** | 50% 葡萄糖靜脈注射 |
| 無法建IV | 昇糖素 1mg 皮下/肌肉注射 |
| ⚠️ 懷疑酒精中毒 | **先 B1，後糖** |

---

# ⚠️ Wernicke 腦病變

<div class="grid grid-cols-2 gap-8">
<div>

**成因**  
長期酗酒 / 營養不良  
→ 慢性 **B1（Thiamine）缺乏**

**陷阱**  
先給葡萄糖 → 消耗殘存 B1  
→ **急性 Wernicke 發作**

**若未治療**  
→ **Korsakoff 精神病**  
（不可逆記憶障礙）

</div>
<div>

### Wernicke Triad

| # | 症狀 |
|---|------|
| 🧠 1 | 意識混亂（躁動、嗜睡） |
| 👁 2 | 眼球運動障礙 / 眼震 |
| 🦵 3 | 步態失調（走路不穩） |

<div class="mt-6 bg-red-100 border-l-4 border-red-600 p-4 rounded-lg text-xl text-center">

**「先 B1，後糖」**

</div>

</div>
</div>
